# AsgardMoe / 神居 Project

## 这是什么？

一个网络游戏，通过 HTTP API （或者使用 HTTP API 的客户端）进行游玩。

## 玩法？

捏人；阅读剧情和进行冒险；或是装扮自己的小窝。你甚至可以理解为这是一个 MUD（Multi-user domain）。

## 社区驱动？

源于社区，回归社区。

- 积极开放 API 接口
- 允许第三方客户端（非作弊用途的）
- 鼓励衍生非商用作品（如同人等）
